# Sistem Pakar Konsultasi Menu Makanan

Project akademik berupa perancangan sistem pakar untuk memberikan rekomendasi menu makanan berdasarkan perhitungan BMI dan kategori pengguna.

## Deskripsi
Sistem ini dirancang untuk membantu pengguna mengetahui rekomendasi menu makanan yang sesuai dengan kondisi tubuh. Sistem menggunakan perhitungan BMI (Body Mass Index) dan aturan (rule-based) sederhana untuk menentukan kategori dan saran menu makanan.

Project ini merupakan tahap perancangan sistem dan belum sampai pada tahap implementasi aplikasi secara penuh.

## Fitur Utama
### Admin
- Mengelola data menu makanan
- Melihat data pengguna yang melakukan konsultasi
- Melihat hasil laporan konsultasi

### Pengguna
- Melihat daftar menu makanan
- Melakukan konsultasi dengan memasukkan berat badan dan tinggi badan
- Mendapatkan hasil perhitungan BMI
- Mendapatkan kategori BMI dan rekomendasi menu makanan

## Konsep yang Digunakan
- Sistem Pakar berbasis rule
- Perhitungan BMI
- CRUD (Create, Read, Update, Delete)
- Multi-user
- Relational Database

## Tools & Teknologi
- Bahasa Pemrograman: Java
- IDE: NetBeans
- Database: MySQL
